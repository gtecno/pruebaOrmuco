Mucho gusto en conocerte ,
Mi nombre es Gabriel Calvo Guzman, en este repositorio encotrará dos archivos :

    1) TestOrmuco-UTB.txt
        
       Este archivo posee toda la información para acceder al servidor
       vía SSH, por favor revisar el archivo y seguir la instrucciones para
       ingresar y poder ejecutar la aplicación python.

	   En el servidor se ha instalado un base de datos MySQL5.5, en el archivo
       se encuntra el usuario y contraseña para el ingreso al servidor de base
	   de datos.
 	
       El aplicativo luego de recolectar los datos, muestra 
       en pantalla los datos alamacenados en base de datos con su identificador.
	   De igual forma, se presenta todos registros almacendos en la base 
	   de datos previamente ingresados.


    2) pruebaOrmuco-UTB.py
	
	   Código fuente para la ejecución del aplicativo en el servidor.



Nota: El servidor se ha desplegado en Windows Azure y corre con una
      distro Linux de UbuntuServer 12.04, en la cual se ha instalado 
      python 2.7, MySQL server 5.5 y conector python-mysql.
